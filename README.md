# 유아체온 모니터링 시스템

# 사용환경
Device : Raspberry Pi 3B+ with WiFi
OS : Raspbian

# 실행 방법
라즈베리파이 화면에서 이 파일을 다운 받은 후 build파일에 GetThermal exe파일을 실행시키면 된다.
