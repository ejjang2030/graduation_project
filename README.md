# 유아체온 모니터링 시스템

# 사용환경
Device : Raspberry Pi 3B+ with WiFi<br>
OS : Raspbian(Linux)

# 실행 방법
build하고 make하면 된다.<br>
라즈베리파이 화면에서 이 파일을 다운 받은 후 build폴더의 release폴더 안에 GetThermal exe파일을 실행시키면 된다.
